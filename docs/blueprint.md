# **App Name**: Hisui Dex Tracker

## Core Features:

- Pokémon List Display: Display all Pokémon from https://pokédex.net/pokedex/game/legends-arceus, including their Hisui Pokédex number, name, and image in a grid or list layout.
- Capture Marking: Allow users to mark each Pokémon as 'captured' using a checkbox or toggle switch.
- Local Storage: Save the 'captured' state of each Pokémon locally in the user's browser.
- Search and Filter: Enable users to filter the Pokémon list by typing a name or number in a search bar.
- Sorting: Allow users to sort the list of Pokémon by number (ascending/descending) and name (A-Z, Z-A).

## Style Guidelines:

- Primary color: Pale Green (#98ff98) evoking nature and freshness.
- Background color: Very Light Green (#edf7ed), near white, providing a clean backdrop.
- Accent color: Desaturated Blue-Green (#66b3ff) for interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern and readable interface.
- Use a clean, modern design with a focus on usability, ensuring an intuitive and easy-to-navigate interface.
- Employ clear and consistent icons to enhance user interaction and understanding.
- Use subtle animations for feedback on interactions such as marking Pokémon as caught or sorting the list.
